# gugu-chat-draft-guard

咕咕助手 - 聊天草稿守护。

这是一个 SillyTavern 第三方扩展，用来给聊天增加“浏览器本地草稿兜底层”。它不修复 VPS 网络，也不替代原生保存逻辑；它解决的是聊天已经出现在前端、但服务端保存失败时，一刷新就可能丢失的问题。

## 功能

- 监听聊天变更并把当前聊天快照写入浏览器本地存储
- 进入同一角色或同一群聊时，如果检测到“本地草稿”和当前已加载聊天不一致，自动提示恢复
- 扩展面板可手动查看、恢复、导出、丢弃本地草稿
- 导出格式为 `jsonl`
- 当原聊天文件丢失时，仍可在同角色的新聊天里恢复最近草稿

## 作用边界

- 只保存在当前浏览器当前设备
- 不做跨设备同步
- 不判断服务器是否一定保存成功，只提供本地保险层

## 安装

1. 在 SillyTavern 的扩展管理里选择从 Git 仓库安装。
2. 填入本仓库地址。
3. 安装后在扩展设置中找到 `咕咕助手 - 聊天草稿守护`。

## 开发

本仓库根目录直接就是第三方扩展目录结构，核心入口如下：

- `manifest.json`
- `index.js`
- `settings.html`
- `style.css`

主要模块：

- `core/`：聊天快照、恢复流程、事件控制
- `storage/`：本地草稿存储和 hash
- `ui/`：设置面板与草稿列表
